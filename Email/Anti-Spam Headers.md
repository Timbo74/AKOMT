## Headers

```
X-Forefront-Antispam-Report:
```
Contains information about the message and about how it was processed.

```
X-Microsoft-Antispam:
```
Contains additional information about bulk mail and phishing.

```
Authentication-results:
```
Contains information about email authentication results including Sender Policy Framework (SPF), Domainkeys Identified Mail (DKIM), and Domain-based Message Authentication, Reporting, and Conformance (DMARC).
