# Shell Folders

|Command|Comment|
|-------|-------|
|`shell:AppsFolder`|Opens the "Applications" folder containing all installed apps.|
|`shell:Startup`|Opens the folder for programs that run automatically at login.|
|`shell:common startup`|Startup for all users.|
|`shell:AppData`|Opens the Roaming AppData folder for user settings.|
|`shell:RecycleBinFolder`|Opens the Recycle Bin.|
|`shell:Downloads`|Directly jumps to your Downloads folder.|
|`shell:Recent`|Opens a list of your recently accessed files|
