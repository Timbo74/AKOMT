# WSL

## List Installed Distros

```
wsl -l
wsl --list
```

## List Available Distros

```
wsl --list --online
```

## Install a Distro
```
wsl --install -d <DistoName>
```

## Remove a Distro
```
wsl --unregister <DisroName>
```

## Set Default Distro

```
wsl --set-default <DistroName>
```

## Launch a Specific Distro

```
wsl -d <DistroName>
```
