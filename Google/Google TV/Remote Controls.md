# Remote Controls

## Using Google TV App

```
Note:
Make sure you are connected to the same Wi-Fi network as the TV.
```

Select the TV from the list near the bottom of the app window, you may be prompted to pair the app on both the app and the TV.  When accepting on the TV you will need to click pair using the TV controls or by using an external keyboard.

Once the TV is selected click on the remote control icon to the right ide of the "Connected - TV NAME" button.

You can now use the oon screen buttons to control your TV.
