# Kitty

**Reference:**
- https://sw.kovidgoyal.net/kitty/

## Config

You can open the **config file** within kitty by pressing ```ctrl+shift+f2```.

### Fonts

Run the ```kitten choose-fonts``` command to use preview fonts and edit the ```kitten.cfg```.
